HTML is the backbone of web development. It helps in structuring and formatting the content of web pages.

What is HTML?

HTML stands for HyperText Markup Language. It is used to design the structure of web pages.

Common Tags in HTML

  1. <b>: For bold text
  2. <i>: For italic text
  3. <u>: For underlined text
  4. <p>: For paragraphs
